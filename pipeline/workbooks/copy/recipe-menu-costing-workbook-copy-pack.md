# Recipe and Menu Costing Workbook Copy Pack

## Workbook Naming

- Primary title: Recipe and Menu Costing Workbook
- Subtitle: Cost every dish clearly, price your menu confidently, and update margins faster when ingredient costs change.

## Start Here Tab

### Welcome note

This workbook helps you calculate what each recipe really costs and what you should charge to protect your margin.

Use it in this order:

1. Set your defaults in `Settings`
2. Add ingredients in `Ingredient Master`
3. Build sub-recipes if needed
4. Cost each dish in `Recipe Costing`
5. Review suggested prices in `Menu Pricing`

### Quick note

Start with one recipe first. Once the structure makes sense, the rest gets much faster.

## Settings Tab

- Section label: Business Defaults
- Helper text: Set your preferred currency, measurement style, and default target food cost percentage here.
- Note: These defaults save time, but you can still override them later for individual recipes.

## Ingredient Master Tab

- Section label: Ingredient Library
- Helper text: Add each ingredient once, then reuse it across every recipe.
- Warning: Make sure your purchase unit and recipe unit match the way you actually buy and use the item.
- Example note: If you buy flour by the kilogram but recipe it in grams, add the conversion so the cost math stays accurate.

## Sub-Recipes Tab

- Section label: Prep Items
- Helper text: Use this tab for sauces, doughs, batters, spice blends, and other reusable prep items.
- Note: A good sub-recipe setup saves time when several menu items use the same prep base.

## Recipe Costing Tab

- Section label: Dish Costing
- Helper text: Build each recipe line by line and let the workbook calculate total batch cost and cost per serving.
- Warning: Check serving count carefully. A small serving error can distort the final cost.

## Menu Pricing Tab

- Section label: Price Guidance
- Helper text: Compare your current selling price against the suggested price based on your target food cost percentage.
- Plain-language note: If your target food cost is `30%`, the selling price needs to be high enough so the ingredient cost only takes up about 30 cents of every dollar sold.

## Summary Dashboard

- Section label: Margin Snapshot
- Helper text: Use this view to spot recipes with weak margins or unusually high ingredient cost.
- Note: This tab is best for quick review after ingredient prices change.

## Light Upgrade Language

- Soft upsell line 1: When this workbook starts feeling manual, `Foodcosting.app` helps you keep ingredient updates, recipes, and pricing in one place.
- Soft upsell line 2: If you are managing many recipes or locations, the app removes a lot of spreadsheet maintenance.
