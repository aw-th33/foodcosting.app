# foodcosting.app Brand Constitution

## Identity
- Product: foodcosting.app
- Audience: Restaurant owners, F&B managers, dark kitchen operators
- Position: The tool that makes food cost visible and actionable

## Color tokens
| Token | Hex | Usage |
|---|---|---|
| Obsidian | #111111 | Headlines, CTAs, final slide background |
| Pure White | #FFFFFF | Always the canvas background |
| Mist | #999999 | Subtext, secondary labels |
| Surface | #F2F2F2 | Ghost numbers, dividers |
| GP Teal | #328589 | Positive margins, profit bars in charts ONLY |
| Loss Red | #E05252 | Negative margins, danger states in charts ONLY |
| Warning Amber | #F0A500 | Caution states in charts ONLY |

GP Teal, Loss Red, and Warning Amber are data-only colors. They never appear as decorative elements, backgrounds, or brand colors in non-chart contexts.

## Typography
- Typeface: Inter (fallback: system-ui, sans-serif)
- Weights: 400 Regular, 500 Medium, 700 Bold, 800 Extrabold
- Scale:
  - Hero: 48px / 800 / -0.03em tracking
  - Headline: 28-32px / 700-800 / -0.02em tracking
  - Subheading: 16-18px / 400 / normal
  - Label: 11px / 500 / 0.1em tracking / UPPERCASE
  - Body: 15px / 400 / 1.6 line-height
  - CTA: 14px / 400 / arrow suffix →
  - Wordmark: 10px / 500 / bottom-right corner

## Layout rules
- Background: #FFFFFF always. No exceptions for social posts.
- Canvas padding: minimum 40px all sides on a 1080x1080 canvas
- Content area: 65-75% of canvas width
- Alignment: centered for social posts
- One primary visual element per slide. Never two competing visuals.
- Ghost numbers: large (120-180px) extrabold, color #F0F0F0, behind all content

## Voice
- Direct, operator-minded, number-first
- Specific over vague: "34% food cost" not "high food cost"
- Never use food puns or startup hype language
- Sentence case everywhere, including headlines
- Preferred formulas:
  - Hook: [Bold claim] + [The number that proves it]
  - Education: [Problem] + [Why it happens] + [Fix]
  - Data: [One number, huge] + [What it means] + [CTA]

## Absolute rules
- Never use colored or gradient backgrounds on social posts
- Never use ALL CAPS or Title Case in headlines
- The black background (#111111) is reserved exclusively for the final CTA slide
- Data accent colors (Teal, Red, Amber) appear only in charts and data visualizations
- Always end carousels with a black CTA slide
